# โมเดล 3 วัน — Flipped Classroom

เนื้อหาเต็มราว 12 โมดูล ส่งมอบใน **3 วัน** โดยเอาทฤษฎีไปไว้ self-study เก็บเวลาสดไว้ลงมือทำ

| ช่วง | เวลา | เนื้อหา |
|------|------|---------|
| **Day 0 — เรียนด้วยตนเอง** | 1 วัน (async) | ดูดซับโมดูล *แนวคิด* แบบเต็ม + ติดตั้งเครื่องมือให้เสร็จก่อนมา |
| **Day 1 + Day 2 เช้า** | 1.5 วัน | เวิร์กช็อปสด: ผู้สอนนำ [[เดโมกลาง — Bean There ระบบจอง\|เดโมกลาง]] + ผู้เรียนทำ [[อภิธานศัพท์#โปรเจกต์ของตัวเอง\|โปรเจกต์ตัวเอง]] คู่ขนาน |
| **Day 2 บ่าย** | 0.5 วัน | Build day: โปรเจกต์ตัวเองล้วน ๆ จนเสร็จ + deploy (lab + ช่วยรายคน) |

## โหมดการส่งมอบของแต่ละเนื้อหา
- 🏠 **self-study เต็ม** — เรียนเองที่บ้าน
- 🗣️ **กล่าวย่อ/ทวนตอนสด** — ผู้สอนทวนสั้น ๆ ในเวิร์กช็อป
- 🛠️ **ลงมือทำสด** — ทำจริงในห้อง

## หลักการจัดสรร
> ทุกแนวคิดเรียนเต็มที่ตอน self-study **แต่ต้องถูกหยิบมาใช้จริง** ในเวิร์กช็อป — ไม่มีทฤษฎีไหนที่ไม่ได้แตะ มิฉะนั้นคนจะลืมและรู้สึกว่าทฤษฎีไม่จำเป็น

## จุดเน้น
**เวิร์กช็อป (ปล่อยของจริง)** — ครึ่งวันท้ายคือ build day ที่ทำให้กลุ่ม A ได้ปล่อยของจริง
