# กลุ่มเป้าหมาย

## กลุ่ม A — เจ้าของไอเดีย/ธุรกิจ (หลัก 🎯)
เจ้าของธุรกิจ / SME / ฟรีแลนซ์ ที่อยากสร้างเว็บหรือแอปของตัวเอง (เว็บร้าน, ระบบจอง, MVP)
- **ไม่ได้อยากเป็นโปรแกรมเมอร์** — อยาก "สร้างของจริงให้เสร็จ"
- **ต้องการ** SDLC / requirement / UX จริง เพราะต้องคิดเองว่าจะสร้างอะไร ให้ใคร
- **ความลึก:** "พอให้ตัดสินใจและสั่ง/ตรวจงาน AI ได้" ไม่ใช่ระดับวิศวกร
- ทำ [[อภิธานศัพท์#โปรเจกต์ของตัวเอง|โปรเจกต์ของตัวเอง]] คู่ขนานกับเดโมกลาง

## กลุ่ม C — คนทั่วไปสายอยากเข้าใจ (รอง)
นักเรียน/นักศึกษา/คนทั่วไป ที่อยากเข้าใจโลกยุค AI และสร้างเป็นบ้าง
- เรียนเพื่อ **เข้าใจ** ไม่เน้นเอาไปทำงาน
- ทำตาม [[เดโมกลาง — Bean There ระบบจอง|เดโมกลาง]] ก็พอ ไม่ต้องมีโปรเจกต์ตัวเอง

## วิธี serve สองกลุ่มพร้อมกัน
ออกแบบความลึกตามกลุ่ม A แล้วให้ C **"เกาะไปด้วย"** ผ่านเดโมกลางตัวเดียวกัน — ทุกแนวคิดผูกกับการลงมือทำ ทำให้ C ไม่หลุดและ A ได้ครบ

> ❌ ไม่เล็งกลุ่ม B (คนอยากเปลี่ยนสายมาเป็น dev) — นั่นต้องการความลึกเชิงวิศวกรรมที่คอร์สนี้จงใจไม่ให้
