# ภาพรวมหลักสูตร

หลักสูตรสอนคน **ไม่มีพื้นฐานการเขียนโปรแกรม** ให้สร้างเว็บแอปได้ด้วย AI ([[อภิธานศัพท์#Vibe Coding|Vibe Coding]]) โดยเชื่อมโยงให้เข้าใจแนวคิดเทคนิค (frontend / backend / database / SDLC / requirement / UX-UI) ในภาษาที่คนไม่มีศัพท์เทคนิคเข้าใจได้

## เป้าหมายเมื่อจบคอร์ส
- มี **เว็บแอปจริง + URL ของตัวเอง** ที่เปิดให้คนทั้งโลกเข้าได้
- เข้าใจภาพรวม frontend / backend / database พอจะ **สั่งและตรวจงาน AI** ได้
- รู้ **กระบวนการสร้างของจริง** (คิด → เก็บ requirement → ออกแบบ → สร้าง → ทดสอบ → ปล่อย)

## หลักการสอน (DNA)
1. **Vibe Coding เป็นแกน** ไม่ใช่ no-code ลากวาง → ผลลัพธ์คือโค้ดจริง deploy ได้
2. **สะพานเชื่อมแนวคิด** — อธิบายเทคนิคผ่านสิ่งที่คุ้นเคย (HTML=โครงบ้าน, Database=ตู้เก็บเอกสาร)
3. **เรียนเพื่อใช้ ไม่ใช่ท่อง** — "อ่านออก ปรับเป็น ก็พอ"
4. **ทุกแนวคิดต้องถูกใช้จริง** ในเวิร์กช็อป ไม่มีทฤษฎีลอย

## เครื่องมือหลัก
- **Antigravity 2.0** — ศูนย์บัญชาการ AI Agent (วางแผน · เขียน · รัน · ทดสอบ)
- **VS Code** — เปิดดู/แก้โค้ดละเอียด
- **Gemini (เว็บ)** — ผู้ช่วยคิด ออกแบบ ร่างเนื้อหา ช่วยเขียน prompt
- **Database:** Google Sheets → Supabase
