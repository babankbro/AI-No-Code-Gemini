# M02 — เครื่องมือ Antigravity 2.0 & Gemini

> 🏠 self-study + 🗣️ ทวนสั้นในห้อง · **ติดตั้งให้เสร็จก่อนมาเวิร์กช็อป**

## สิ่งที่จะได้เรียนรู้
- Antigravity เป็นแพลตฟอร์มแบบ Agent-first
- ระบบนิเวศ 4 ส่วนของเวอร์ชัน 2.0
- ฟีเจอร์เด่น + การเลือกโมเดล AI
- บทบาท Gemini และ Dual-Wield Workflow
- สเปคเครื่อง + แพ็กเกจ Free/Pro/Ultra

## แนวคิดหลัก
**Antigravity** = แพลตฟอร์มพัฒนาแบบ Agent-first (พัฒนาโดย Google, ต่อยอดจาก VS Code) — AI Agent ลงมือเขียนโค้ด รันใน Terminal และทดสอบในเบราว์เซอร์ได้เอง โดยมีเราเป็นผู้ควบคุม

**2 วิธีทำงาน:**
- **Editor View** — เขียนเอง + AI ช่วยเติม (คุมละเอียด)
- **Agent Manager** — มอบงานทั้งก้อนให้ Agent ทำเอง (รันขนานได้สูงสุด 5)

**Artifacts** — ผลงานที่ "คนตรวจได้": Plans / Task Lists / Screenshots / Recordings

**เลือกโมเดลได้:** Gemini 3 Pro (วางแผน/งานซับซ้อน) · Gemini 3 Flash (เร็ว/ประหยัด) · Claude Sonnet 4.6 (เขียนโค้ดเก่ง) · Claude Opus 4.6 (งานยาก) · GPT-OSS

## Dual-Wield Workflow (เครื่องมือ 3 ตัวร่วมกัน)
- **Antigravity 2.0** — ศูนย์บัญชาการ Agent (สั่ง · เขียน · รัน · ทดสอบ)
- **VS Code** — เปิดดู/แก้โค้ดละเอียด
- **Gemini (เว็บ)** — ผู้ช่วยคิด ออกแบบ ร่างเนื้อหา ช่วยเขียน prompt

## เตรียมตัวก่อนมา (สำคัญ!)
- ดาวน์โหลดฟรีที่ antigravity.google (ไม่ต้องใช้บัตรเครดิต)
- สเปค: Windows 10+/macOS 12+/Linux · RAM 8GB+ (แนะนำ 16GB) · พื้นที่ 10GB+
- ⚠️ Free tier จำกัด ~20 ครั้ง/วัน — workshop 20–30 คนอาจชนลิมิต แนะนำเตรียมบัญชี Pro ส่วนกลาง
