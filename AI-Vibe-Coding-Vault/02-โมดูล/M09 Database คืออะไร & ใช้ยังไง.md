## ความลึกระดับ 2 (ไม่มากไม่น้อย)
รู้: ตาราง / แถว / คอลัมน์ / CRUD / **ความสัมพันธ์เบื้องต้น**

## สะพานเชื่อมแนวคิด: ตู้เก็บเอกสาร
| คำเทคนิค | เปรียบเหมือน |
|---------|-------------|
| Database | ตู้เก็บเอกสารทั้งตู้ |
| ตาราง (Table) | ลิ้นชัก 1 ช่อง (เก็บของชนิดหนึ่ง เช่น "การจอง") |
| แถว (Row) | แฟ้ม 1 ใบ (1 รายการ เช่น การจอง 1 ครั้ง) |
| คอลัมน์ (Column) | ช่องในแฟ้ม (1 ข้อมูล เช่น ชื่อ, วันจอง) |
| CRUD | เพิ่ม / ดู / แก้ / ลบ |

> ต่างจาก Excel: database เก็บถาวร + หลายคนใช้พร้อมกัน + เว็บดึงไปแสดงสดได้
