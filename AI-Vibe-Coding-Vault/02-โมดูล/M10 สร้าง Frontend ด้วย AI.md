# M10 — สร้าง Frontend ด้วย AI 🆕

> 🛠️ ลงมือทำสด · **Day 1** · จุดเน้นของคอร์ส

## เป้าหมาย
สร้างหน้าเว็บจริงของ [[เดโมกลาง — Bean There ระบบจอง|เดโมกลาง]]: หน้าเมนู + ฟอร์มจอง จาก [[M06 ออกแบบ UX-UI เบื้องต้น|wireframe]] ที่ออกแบบไว้

## ลำดับลงมือ
1. หยิบ wireframe (ภาพถ่าย) + [[M05 เก็บ Requirement ให้ชัด|SRS]] มาเป็นวัตถุดิบ
2. เขียน prompt ตาม [[M03 การเขียน Prompt|สูตร C-G-R-E-F]] สั่ง AI สร้างหน้าเมนู
3. สั่ง AI เพิ่มฟอร์มจอง (ชื่อ, เบอร์, วัน/เวลา, จำนวนคน)
4. สั่ง "ให้ responsive" → AI จัดการทุกจอ
5. ปรับแต่งหน้าตา/สีกับ AI+Gemini (เราไม่ต้องรู้ทฤษฎีสี)

## โหมดคู่ขนาน
- **กลุ่ม A** ทำ frontend ของ [[อภิธานศัพท์#โปรเจกต์ของตัวเอง|โปรเจกต์ตัวเอง]] ตามแพตเทิร์นเดียวกัน
- **กลุ่ม C** ทำตามเดโมกลาง
