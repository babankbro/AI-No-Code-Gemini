# 📁 AI Vibe Coding Vault (Obsidian)

นี่คือ **Obsidian vault** ของหลักสูตร "AI Vibe Coding สำหรับคนไม่มีพื้นฐาน"

## วิธีเปิด
1. ติดตั้ง [Obsidian](https://obsidian.md) (ฟรี)
2. เปิด Obsidian → **Open folder as vault** → เลือกโฟลเดอร์ `AI-Vibe-Coding-Vault`
3. เริ่มที่ [[00-Home]] แล้วกดตาม `[[ลิงก์]]` หรือดู **Graph View** เพื่อเห็นการเชื่อมโยง

## โครงสร้าง
```
AI-Vibe-Coding-Vault/
├── 00-Home.md                 ← เริ่มที่นี่ (Map of Content)
├── 01-ภาพรวม/                  ← กลุ่มเป้าหมาย, โมเดล 3 วัน, เส้นทาง 6 ขั้น, ขอบเขต
├── 02-โมดูล/                   ← M01–M12 (เนื้อหาทุกโมดูล)
├── 03-เดโม & Asset/            ← เดโมกลาง, พรอมต์ grill-me, Requirement Sheet
└── 04-อ้างอิง/                 ← อภิธานศัพท์
```
