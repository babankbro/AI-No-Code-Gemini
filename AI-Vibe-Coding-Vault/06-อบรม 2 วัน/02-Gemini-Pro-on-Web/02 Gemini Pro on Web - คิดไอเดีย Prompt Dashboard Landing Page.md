# 02 Gemini Pro on Web - คิดไอเดีย Prompt Dashboard Landing Page

## เป้าหมายของช่วงนี้

- ให้ผู้เรียนใช้ Gemini เป็นผู้ช่วยคิดก่อนเริ่มสร้างเว็บ
- เปลี่ยนไอเดียกว้าง ๆ ให้เป็น requirement และ prompt ที่ใช้ได้
- แสดงให้เห็นว่า Gemini ช่วยทั้งข้อความหน้าเว็บและแนวคิด dashboard ได้

## สิ่งที่ผู้เรียนต้องเปิด

- [Gemini](https://gemini.google.com/)
- ไฟล์ CSV ตัวอย่าง หรือข้อมูลที่เตรียมไว้
- หน้าจด prompt ของตัวเอง

## Flow การสอน 60 นาที

| นาที | เนื้อหา |
|---|---|
| 0-10 | อธิบายบทบาท Gemini: คิด, ถาม, สรุป, ช่วยเขียน prompt |
| 10-20 | Demo: ให้ Gemini แตกไอเดีย landing page |
| 20-35 | Hands-on: ผู้เรียนใส่ไอเดียของตัวเอง แล้วให้ Gemini ถามต่อ |
| 35-45 | Demo: ให้ Gemini วิเคราะห์ CSV และเสนอ dashboard |
| 45-55 | ให้ Gemini เขียน prompt สำหรับส่งต่อ Antigravity |
| 55-60 | สรุป output ที่ต้องได้ก่อนพัก |

## Demo 1: แตกไอเดียเป็น Requirement

ใช้ prompt:

```text
ฉันอยากสร้างเว็บสำหรับ [อธิบายไอเดียสั้น ๆ]

ช่วยถามคำถามเพื่อทำให้ไอเดียนี้ชัดขึ้น โดยถามทีละข้อ
ให้ครอบคลุม:
- ใครคือผู้ใช้
- ผู้ใช้ต้องการทำอะไร
- ฟีเจอร์ที่จำเป็นที่สุดคืออะไร
- ข้อมูลที่ระบบต้องเก็บคืออะไร
- หน้าเว็บควรมีกี่หน้า

เมื่อฉันพิมพ์ว่า "สรุป" ให้สรุปเป็น requirement สั้น ๆ สำหรับใช้สั่ง AI สร้างเว็บ
```

## Demo 2: สร้าง Prompt สำหรับ Landing Page

ให้ Gemini สรุปออกมาเป็น:

- ชื่อเว็บ/ชื่อธุรกิจ
- กลุ่มเป้าหมาย
- เป้าหมายของหน้าเว็บ
- Section ที่ต้องมี
- tone & style
- call-to-action

## Demo 3: วิเคราะห์ CSV เป็น Dashboard

ให้ Gemini ดูโครงสร้างข้อมูล แล้วตอบ:

- คอลัมน์ไหนสำคัญ
- ควรมี summary card อะไร
- ควรใช้ chart แบบไหน
- ผู้ใช้ dashboard ต้องตอบคำถามอะไรได้

## Hands-on ที่ผู้เรียนต้องทำ

- [ ] พิมพ์ไอเดียของตัวเองใน Gemini
- [ ] ให้ Gemini ถามต่ออย่างน้อย 5 คำถาม
- [ ] พิมพ์ "สรุป" แล้วเก็บ requirement
- [ ] ขอ prompt สำหรับสร้าง landing page
- [ ] ขอแนวคิด dashboard จาก CSV

## Output ก่อนจบช่วง

- Prompt สร้าง landing page
- Requirement สั้น ๆ
- แนวคิด dashboard 1 ชุด
