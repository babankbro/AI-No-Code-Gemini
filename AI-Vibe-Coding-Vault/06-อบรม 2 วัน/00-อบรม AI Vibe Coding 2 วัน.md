# อบรม AI Vibe Coding 2 วัน

เวิร์กช็อป 2 วันสำหรับพาผู้เรียนจาก "ใช้ AI ช่วยคิด" ไปถึง "สร้างเว็บแอปและ deploy ได้จริง" โดยใช้ Gemini, Antigravity, GitHub, Vercel, Google Sheets, API และ Supabase

## เป้าหมาย

- เข้าใจว่า [[อภิธานศัพท์#Vibe Coding|Vibe Coding]] คือการสั่ง AI ให้สร้างโค้ดจริง ไม่ใช่ no-code ลากวาง
- ใช้ Gemini ช่วยคิดไอเดีย เขียน prompt วิเคราะห์ข้อมูล และออกแบบหน้าเว็บ
- ใช้ Antigravity สร้าง landing page, dashboard และเว็บที่ต่อข้อมูลได้
- ใช้ Git, GitHub และ Vercel เพื่อ deploy เว็บขึ้นออนไลน์
- เข้าใจเส้นทางข้อมูล: CSV -> Google Sheets -> API -> Supabase

## ภาพรวม 2 วัน

| วัน | แกนหลัก | ผลลัพธ์ปลายวัน |
|---|---|---|
| วันที่ 20 | Motivation -> Gemini -> Antigravity -> Landing Page -> Git/GitHub/Vercel | ผู้เรียนมี landing page และพร้อม deploy |
| วันที่ 21 | UI/UX -> Google Sheets -> Dashboard -> API -> Supabase -> Vercel | ผู้เรียนมีเว็บ/แดชบอร์ดที่เชื่อมข้อมูลและมี URL จริง |
