# โมดูล 02
## รู้จัก Antigravity 2.0 & Gemini แบบเจาะลึก

ศูนย์บัญชาการ AI Agent · เครื่องมือหลักของหลักสูตร

> 🎤 ก่อนเริ่ม: เช็กว่าผู้เรียนติดตั้ง Antigravity มาแล้วหรือยัง (โมดูลนี้ต้องลงมือก่อนมาเวิร์กช็อป)

---

## สิ่งที่จะได้เรียนรู้

- Antigravity เป็นแพลตฟอร์มแบบ Agent-first
- ระบบนิเวศ 4 ส่วนของเวอร์ชัน 2.0
- ฟีเจอร์เด่น + การเลือกโมเดล AI
- บทบาท Gemini และ Dual-Wield Workflow
- สเปคเครื่อง + แพ็กเกจ Free/Pro/Ultra

---

## Antigravity คืออะไร?

- 🤖 แพลตฟอร์มพัฒนาแบบ **Agent-first** — ไม่ใช่แค่ Editor
- AI Agent ลงมือ **เขียนโค้ด · รันใน Terminal · ทดสอบในเบราว์เซอร์** เองได้ (เราคุม)
- 🏢 พัฒนาโดย Google (เปิดตัว พ.ย. 2025 พร้อม Gemini 3)
- 🧩 ต่อยอดจาก VS Code — หน้าตาคุ้นเคย

---

## ระบบนิเวศ 4 ส่วน (เวอร์ชัน 2.0)

1. **Antigravity IDE** — เอดิเตอร์ (fork จาก VS Code) + Agent Manager
2. **Desktop App 2.0** — ศูนย์บัญชาการ Agent ← *ตัวที่เราใช้ในคอร์ส*
3. **Antigravity CLI** — สั่ง Agent ผ่าน Terminal
4. **Antigravity SDK** — เชื่อมเข้าระบบของเราเอง

---

## 2 วิธีทำงาน

| Editor View | Agent Manager |
|-------------|---------------|
| เขียนเอง + AI ช่วยเติม (Tab) | มอบงานทั้งก้อนให้ Agent |
| สั่งด้วยภาษาคนในบรรทัด | รันหลาย Agent ขนาน (สูงสุด 5) |
| เหมาะงานคุมละเอียด | เหมาะงานใหญ่/ซ้ำ ๆ |
| เหมือน VS Code ที่ฉลาดขึ้น | เราเป็นผู้ควบคุม+ตรวจผล |

---

## ฟีเจอร์เด่นเวอร์ชัน 2.0

- 🔀 **Sub-Agents ขนาน** — รันหลายตัวพร้อมกัน
- 🌐 **Browser-in-the-loop** — ทดสอบในเบราว์เซอร์จริง
- 📦 **Artifacts** — ผลงานที่ตรวจสอบได้
- 🧠 **Knowledge Base** — จดจำ context โปรเจกต์
- ⏰ **Scheduled Tasks** — ตั้งเวลาให้ Agent
- 🔄 **Multi-model** — เลือกโมเดลได้หลายตัว

---

## "Artifacts" — ผลงานที่ตรวจได้

แทนที่จะให้เราอ่าน log ยาว ๆ Agent สรุปงานเป็นชิ้นที่ "คนตรวจได้":

- 📋 **Plans** — ขั้นตอนที่ Agent จะทำ
- ✅ **Task Lists** — งานย่อยและสถานะ
- 📸 **Screenshots** — ภาพผลลัพธ์
- 🎥 **Recordings** — คลิปการทำงานในเบราว์เซอร์

> 🎤 จุดนี้สำคัญสำหรับคนไม่มีพื้น: เราตรวจงาน AI ผ่าน "ของที่ดูรู้เรื่อง" ไม่ใช่โค้ดดิบ

| โมเดล | จุดเด่น |
|-------|---------|
| Gemini 3 Pro | วางแผน/งานซับซ้อน (ตัวหลัก) |

---

## Gemini & บทบาทในคอร์ส

Gemini = AI ของ Google (ใช้ผ่าน gemini.google.com) — ผู้ช่วยคิดแยกจาก Antigravity

ในคอร์สใช้ Gemini เพื่อ:
- 💡 คิดไอเดีย + วางโครงสร้างเว็บ
- ✍️ ร่างเนื้อหา/ข้อความบนเว็บ
- 🎯 ช่วยเขียน Prompt ให้คมขึ้น
- 🎨 ปรึกษาดีไซน์และโทนสี

---

## Dual-Wield Workflow — 3 เครื่องมือร่วมกัน

| Antigravity 2.0 | VS Code | Gemini (เว็บ) |
|-----------------|---------|---------------|
| ศูนย์บัญชาการ Agent | เปิดดู/แก้โค้ดละเอียด | ผู้ช่วยคิด & ออกแบบ |
| สั่ง·เขียน·รัน·ทดสอบ | ตรวจไฟล์·จัดการโปรเจกต์ | ไอเดีย·เนื้อหา·Prompt·ดีไซน์ |

> สลับใช้ทั้ง 3 ลื่นไหล — แต่ละตัวทำสิ่งที่ตัวเองเก่งสุด

## สเปคเครื่อง & การติดตั้ง

- 💻 OS: Windows 10+ · macOS 12+ · Linux
- 🧠 RAM: 8GB+ (แนะนำ 16GB)
- 💾 พื้นที่: 10GB+
- ⬇️ ดาวน์โหลดฟรี: **antigravity.google** (ไม่ต้องใช้บัตรเครดิต)

## แพ็กเกจ: Free · Pro · Ultra

| Free ($0) | Pro ($20/ด.) | Ultra ($249.99/ด.) |
|-----------|--------------|---------------------|
| ทุกโมเดล (มี rate limit) | ลิมิตสูงขึ้น | ลิมิตสูงสุด งานหนัก |
| รีเฟรช ~5 ชม. | ประมวลผลก่อน | เซสชัน Agent ยาว |
| เหมาะทดลอง | คุ้มงานทั่วไป | สำหรับทีม/มืออาชีพ |

> ⚠️ **ทิป Workshop:** Free ~20 ครั้ง/วัน — 20–30 คนพร้อมกันอาจชนลิมิต เตรียมบัญชี Pro ส่วนกลาง
