# คู่มือการเตรียมความพร้อมสำหรับผู้อบรม (Workshop Setup Guide)

เนื่องจากการทำ Web Dashboard ที่มีการดึงข้อมูลจริง (Fetch API) ข้ามเว็บไซต์ เช่น จาก Google Sheets เบราว์เซอร์ยุคใหม่ (Chrome, Edge, Safari) จะมีระบบรักษาความปลอดภัย **CORS (Cross-Origin Resource Sharing)** ทำให้การดับเบิลคลิกเปิดไฟล์ `index.html` ตรงๆ (ซึ่งจะรันผ่าน `file:///`) จะถูกบล็อกและดึงข้อมูลไม่ได้

เพื่อให้ผู้อบรมสามารถรันโค้ดและเห็นผลลัพธ์ได้อย่างสมบูรณ์ ขอแนะนำ 2 วิธีที่ง่ายและเป็นที่นิยมที่สุดครับ:

---

## 🛠️ วิธีที่ 1: ใช้โปรแกรม VS Code + Live Server (⭐️ แนะนำสำหรับผู้เริ่มต้น / No-Code)
วิธีนี้ง่ายที่สุด ไม่ต้องพิมพ์คำสั่งใน Terminal เหมาะสำหรับผู้อบรมทุกระดับ

**ขั้นตอน:**
1. **ดาวน์โหลดและติดตั้ง VS Code**
   - ไปที่เว็บ: [code.visualstudio.com](https://code.visualstudio.com/)
   - กดปุ่ม Download และติดตั้งแบบ Next ไปเรื่อยๆ จนเสร็จ
2. **ติดตั้ง Extension "Live Server"**
   - เปิดโปรแกรม VS Code ขึ้นมา
   - กดที่เมนู Extensions แถบด้านซ้ายมือ (ไอคอนรูปกล่อง 4 ใบ) หรือกดปุ่มลัด `Ctrl + Shift + X`
   - พิมพ์ค้นหาคำว่า `Live Server` (พัฒนาโดย Ritwick Dey)
   - กดปุ่ม **Install**
3. **การใช้งานเปิด Dashboard**
   - ลากโฟลเดอร์โปรเจกต์ `demo_dashboard` (ที่มีไฟล์ `index.html`) มาปล่อยในหน้าจอ VS Code
   - คลิกเปิดไฟล์ `index.html`
   - มองไปที่มุมขวาล่างของโปรแกรม VS Code จะเห็นปุ่ม **"Go Live"** ให้คลิก 1 ครั้ง
   - เบราว์เซอร์จะเด้งขึ้นมาเองที่ URL เช่น `http://127.0.0.1:5500/index.html` และโค้ดจะรันได้ 100% ไม่มีปัญหาการบล็อกครับ!

---

## 💻 วิธีที่ 2: ใช้ Node.js (สำหรับผู้ที่พอใช้ Terminal เป็น)
วิธีนี้เป็นการรัน Local Server ผ่านคำสั่งโดยตรง เหมาะสำหรับผู้ที่มีพื้นฐานสาย Developer หรืออยากลองใช้ Command Line

**ขั้นตอน:**
1. **ดาวน์โหลดและติดตั้ง Node.js**
   - ไปที่เว็บ: [nodejs.org](https://nodejs.org/en/)
   - ดาวน์โหลดเวอร์ชัน **LTS (Long Term Support)** และติดตั้งตามปกติ (กด Next รัวๆ)
2. **ตรวจสอบการติดตั้ง**
   - เปิดโปรแกรม Terminal, Command Prompt (cmd) หรือ PowerShell
   - พิมพ์คำสั่ง `node -v` (ถ้าติดตั้งสำเร็จ จะขึ้นตัวเลขเวอร์ชัน เช่น v20.x.x)
3. **การใช้งานเปิด Dashboard**
   - เปิด Terminal แล้วใช้คำสั่ง `cd` เข้าไปยังโฟลเดอร์โปรเจกต์ ตัวอย่างเช่น:
     ```bash
     cd path/to/demo_dashboard
     ```
   - พิมพ์คำสั่งรัน Server ด้านล่างนี้ แล้วกด Enter:
     ```bash
     npx http-server -p 8000
     ```
   - จากนั้นให้ผู้เรียนเปิดเบราว์เซอร์ แล้วพิมพ์ URL ไปที่ 👉 `http://localhost:8000`

---

> [!TIP]
> **💡 เทคนิคเพิ่มเติมสำหรับวิทยากร:**
> หากต้องการใช้ Prompt เพื่อให้ AI สร้างคำแนะนำการแก้ปัญหาให้ผู้เรียน สามารถให้นักเรียนพิมพ์ Prompt บอก AI ได้ดังนี้:
> *"ช่วยแนะนำวิธีจำลอง Local Server อย่างง่าย เพื่อเปิดไฟล์ index.html ให้หน่อย เพราะตอนนี้ติดปัญหา CORS เวลาดึงข้อมูลผ่าน Fetch API (ขอวิธีที่ไม่ซับซ้อน เช่น ใช้ Extension ของ VS Code หรือ Node.js)"*
