# ยึด Vibe Coding (AI เขียนโค้ดจริง) ไม่ใช่ No-code แท้

แม้ผู้เรียนไม่มีพื้นฐานและคำว่า "no-code" จะการตลาดดีกว่า เราเลือกสอน **Vibe Coding** — สั่งเป็นภาษาคนแล้ว AI เขียนโค้ดจริงที่ deploy ได้ — แทนเครื่องมือ no-code ลากวาง (Bubble/Webflow) เพราะ (1) วัสดุเดิมทั้งหมดเป็นแนวนี้, (2) เป้าหมายคือให้ผู้เรียน "เข้าใจ frontend/backend/database จริง" ซึ่ง no-code แท้ซ่อนสิ่งเหล่านี้ไว้หมด, (3) ทักษะโอนต่อยอดได้กว้างกว่า. คำว่า "no-code" ใช้ได้เฉพาะเชิงการตลาด ("สร้างได้โดยไม่ต้องเขียนเอง") ห้ามใช้แทน Vibe Coding ในเนื้อหา.
