# กลุ่มเป้าหมาย A+C และโมเดล flipped classroom 3 วัน

เล็งกลุ่ม **A (เจ้าของไอเดีย/ธุรกิจ อยากสร้างของจริง) เป็นหลัก + C (คนทั่วไปสายอยากเข้าใจ) เป็นรอง** โดยออกแบบความลึกตามกลุ่ม A ("พอให้ตัดสินใจและสั่ง/ตรวจงาน AI ได้ ไม่ใช่ระดับวิศวกร") แล้วให้ C เกาะไปด้วยผ่านเดโมกลางเดียวกัน.

เนื้อหาเต็มราว 12 โมดูลส่งมอบใน **3 วันแบบ flipped**: 1 วันเรียนแนวคิดด้วยตนเอง (async) + 1.5 วันเวิร์กช็อปสด (เดโมกลาง + โปรเจกต์ตัวเองคู่ขนาน) + 0.5 วัน build โปรเจกต์ตัวเองล้วน. เลือกแบบนี้แทน "ยัดลง 2 วันเดิม" (จะตื้นเกิน) และแทน "แยก 2 คอร์ส" (commitment กระจาย) เพราะเก็บเวลาสดไว้ลงมือทำ — ตรงจุดเน้นที่ว่าจบแล้วต้องมีของจริง + URL.

**ผลที่ตามมา:** ทุกแนวคิดต้องถูกหยิบมาใช้จริงในเวิร์กช็อป มิฉะนั้น self-study จะถูกมองว่าไม่จำเป็น.
